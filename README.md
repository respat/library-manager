
## Projekt telepítése
```
npm run setup-project
```

## Projekt indítása
```
npm run dev
```

## Az adatbázishoz szükséges .env fájl
```
NODE_ENV=development
PORT=5100
MONGO_URL=mongodb+srv://databaseUser:password@cluster0.example.mongodb.net/?retryWrites=true&w=majority
JWT_SECRET=secret
JWT_EXPIRES_IN=1d
```
